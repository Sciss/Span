### linking

    libraryDependencies += "de.sciss" %% "span" % v

The current version `v` is `"1.2.+"`

## building

This project builds against Scala 2.10, using sbt 0.12. It uses the [LucreSTM](https://www.github.com/Sciss/LucreSTM) module `serial`.
